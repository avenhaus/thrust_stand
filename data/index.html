Sensor Calibration
About Calibration: Calibration values adjust how raw sensor readings are converted to meaningful units (grams, amps). Use the INA226 Scan to find recommended zero offset, or manually enter values if you have known references. Changes apply immediately.
Thrust/Torque calibration instructions:
1. Tare the sensors with no load attached.
2. Place a known weight on the thrust sensor and adjust the Thrust factor until the reading matches the expected load.
3. Place a known torque reference or use a calibrated test mass on the torque arm and adjust the Torque factor.
4. Use the INA226 scan to capture a zero-current offset with the motor off and no load.
5. Save the values and verify by re-testing with the same known references.
